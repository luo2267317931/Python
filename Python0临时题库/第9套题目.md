## 任务3：

编程实现判断一个字符串是否是“回文串”。所谓“回文串”是指一个字符串的第一位与最后一位相同，第二位与倒数第二位相同。

例如：“159951”、 “19891”是回文串，而“2011”不是。

要求：用带有一个输入参数的函数(或方法)实现，返回值类型为布尔类型。
1.第一种方法
```python
#!/usr/bin/python 
# -*- coding: utf-8 -*-
s = input("请输入一个回文数")
j = len(s) - 1
for i in range(0, len(s), 1):
    if s[i] == s[j]:
        j -= 1
    else:
        print(f"{s}不是回文数字")
        break
if j == -1:
    print("是")
```
